# Metric Policy

## Impact Factor

Only report impact factor when it can be reliably verified from a trusted source available in the current environment.
If it cannot be verified, write `недостъпно`.

## Citation Counts

Treat citation counts as dynamic.
Use them as supportive evidence, not as the only ranking signal.

## Influence

Estimate influence using a combination of:
- citation-informed evidence
- review status
- venue credibility
- topical centrality
- recency

Do not rank only by raw citation count.

## Missing Metrics

If a metric cannot be verified:
- do not guess
- do not infer an exact number
- explain the limitation briefly
