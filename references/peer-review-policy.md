# Peer-Review Policy

## Include In The Main Corpus

- peer-reviewed journal articles
- review articles
- systematic reviews
- meta-analyses
- strong original research papers

## Usually Exclude

- preprints
- editorials
- conference abstracts
- news items
- patents
- theses unless the user explicitly asks for them

## Exceptions

- If the user explicitly asks for frontier or emerging work, include preprints in a separate section labeled `Non-peer-reviewed context`.
- If peer-review status is ambiguous, mark it as `unclear` rather than assuming.
